# react-naver-maps

React 라이브러리 프로젝트입니다.

## 설치

```bash
npm install react-naver-maps
# 또는
yarn add react-naver-maps
# 또는
pnpm add react-naver-maps
```

## 사용법

```tsx
import { ExampleComponent } from 'react-naver-maps';

function App() {
  return (
    <ExampleComponent 
      text="Hello World"
      onClick={() => console.log('Clicked!')}
    />
  );
}
```

## 개발
