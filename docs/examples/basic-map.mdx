# 기본 지도 예제

가장 기본적인 지도 표시 예제입니다.

## 라이브 예제

<BrowserOnly>{() => <BasicMapExample />}</BrowserOnly>

## 코드

```tsx
import { Map } from "@rousen/react-naver-maps";

function BasicMap() {
  return (
    <div style={{ width: "100%", height: "400px" }}>
      <Map ncpKeyId="your-ncp-key-id" />
    </div>
  );
}

export default BasicMap;
```

## 설명

- `Map` 컴포넌트에 `ncpKeyId` prop만 전달하면 기본 지도가 표시됩니다
- 지도 컨테이너에 `width`와 `height` 스타일을 지정해야 지도가 보입니다
- 기본적으로 서울시청 근처가 중심점으로 설정됩니다
