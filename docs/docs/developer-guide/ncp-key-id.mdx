# ncpKeyId 발급

`ncpKeyId`는 네이버 지도 API를 사용하기 위한 클라이언트 ID입니다.  
지도 렌더링 시 `Map` 컴포넌트의 `ncpKeyId` prop으로 전달합니다.

```tsx
<Map ncpKeyId="your-ncp-key-id" />
```

## 발급 절차

1. [네이버 클라우드 플랫폼](https://www.ncloud.com/)에 가입합니다
2. 콘솔에서 **AI·NAVER API** > **Maps** 메뉴로 이동합니다
3. **Application 등록**을 통해 클라이언트 ID를 발급받습니다
4. 발급받은 클라이언트 ID를 `ncpKeyId`로 설정합니다

## 운영 시 주의사항

- 발급한 키는 외부에 노출되지 않도록 관리합니다.
- 운영 환경에서는 도메인 제한 등 보안 설정을 적용하는 것을 권장합니다.
