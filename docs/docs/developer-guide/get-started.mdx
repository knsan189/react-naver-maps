# Get Started

`@rousen/react-naver-maps`를 설치하고 가장 빠르게 지도를 렌더링하는 방법을 정리합니다.

## 1. 설치

```bash
npm install @rousen/react-naver-maps
```

TypeScript에서 `naver.maps.*` 전역 타입을 쓰려면 `@types/navermaps`를 개발 의존성으로 추가하세요.

```bash
npm install -D @types/navermaps
```

## 2. 지도 렌더링

```tsx
function App() {
  return <Map ncpKeyId="your-ncp-key-id" />;
}
```

import BrowserOnly from "@docusaurus/BrowserOnly";
import { Map } from "@rousen/react-naver-maps";

<BrowserOnly>{() => <BasicMapExample />}</BrowserOnly>

## 참고

- `ncpKeyId`는 네이버 클라우드 플랫폼 콘솔에서 발급한 클라이언트 ID입니다.
- 더 많은 예제는 [예제 보기](/examples/basic-map)에서 확인할 수 있습니다.
